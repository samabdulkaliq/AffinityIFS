# **App Name**: Affinity

## Core Features:

- Secure Access & Role-Based Dashboards: User authentication for Cleaner and Admin roles. Presents tailored dashboards displaying relevant information based on the user's role.
- Web Clock-In/Out & Automated Time Events: Cleaners can digitally 'Arrive' and 'Depart' from job sites, simulating a geofence trigger. The system automatically generates CLOCK_IN, BREAK_START, BREAK_END, and CLOCK_OUT events, applying the Ontario break rule where applicable.
- Time Review & Approval Workflow: Cleaners submit requests for time adjustments (e.g., missed clock-outs, disputes, exceptions). Admins manage an approval queue to review, approve, or reject these requests with adjustment logs.
- Intelligent Time Review Assistant: An AI tool that assists administrators by analyzing time review requests, highlighting discrepancies, and suggesting potential resolutions or categorizations to streamline the approval process.
- Gamification: Points & Leaderboard: Track and display cleaner points for punctuality, exception-free shifts, and ratings. Features a 'Points Wallet' and a monthly 'Leaderboard', with UI placeholders for rewards redemption.
- In-App Notification Center: A centralized hub for categorized notifications (e.g., Time & Attendance, Approvals, Reminders, Rewards) with friendly, context-aware microcopy and light emoji use for both cleaners and admins.

## Style Guidelines:

- Primary color: A professional, deep charcoal grey (#4A4A4A), reflecting the strong text of the Affinity logo, providing a solid, trustworthy foundation.
- Accent color: A vibrant, clean cyan (#00BCD4), chosen from the bright spectrum of the Affinity logo, to highlight interactive elements, calls to action, and bring a touch of dynamic energy.
- Secondary color palette: The full rainbow spectrum of the Affinity logo (e.g., various blues, greens, yellows, oranges, reds, purples) will be utilized subtly for data visualization, progress indicators, and to add diverse, playful accents without overwhelming the professional feel.
- Background color: A serene, ultra-light cool grey with a hint of blue (#E8EEF4), ensuring a clean and spacious feel, allowing the primary and accent colors to stand out effectively.
- Headlines and body text: 'Inter', a modern sans-serif that offers excellent readability and a contemporary, objective aesthetic suitable for a professional platform.
- Utilize simple, clean line-art icons that maintain a modern, uncluttered look. Incorporate light emoji usage specifically within notification messages for a friendly tone.
- Card-based layout featuring rounded corners, subtle shadows, and consistent spacing to promote a premium and organized visual experience. Employ subtle gradients sparingly for depth and visual interest on key UI elements.
- Implement subtle transition animations between dashboard views and form submissions, ensuring a smooth and responsive user experience. Loading skeletons should be used to elegantly handle data fetching states.
